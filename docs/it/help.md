# Riferimento Aiuto

Questa pagina contiene il contenuto di aiuto integrato nell'applicazione.

## Gruppo di Acquisto

Selezionare il gruppo di acquisto che si desidera osservare.

## Responsabile Autore delle Dichiarazioni

Seleziona un responsabile tra quelli del gruppo di acquisto selezionato.

Le dichiarazioni finora pubblicate dal responsabile vengono caricate in sessione per permettere la loro eliminazione e la aggiunta di nuove.

## Gestione Dichiarazioni

Questo menu permette di aggiungere nuove dichiarazioni alla sessione.

Inoltre nell'ultima parte si trovano alcune operazioni gestionali che riguardano le dichiarazioni stesse:

- **La pubblicazione delle dichiarazioni** sostituisce le dichiarazioni precedentemente pubblicate dal responsabile selezionato con quelle attualmente in sessione
- **L'eliminazione delle dichiarazioni** elimina le dichiarazioni in sessione. La successiva pubblicazione corregge quelle pubblicate
- **La correzione del livello di considerazione** delle dichiarazioni permette di osservare un caricamento in bianco incompleto. Questo risulta necessario quando vogliamo produrre una dichiarazione che ha diritto di essere pubblicata, ma il caricamento di tutte le dichiarazioni impedisce al sistema di proporcela

## Effetto delle Ultime Dichiarazioni

Un calcolo *in bianco* degli effetti delle dichiarazioni sullo stato corrente.

Le dichiarazioni inserite comprendono:

- Le dichiarazioni pubblicate di tutti i responsabili tranne quello selezionato
- Le dichiarazioni in sessione del responsabile selezionato

## Amministrazione

Menu di amministrazione per il gestore del servizio e per ottenere delle nuove chiavi.

## Inserimento Password

La password serve a nascondere la parte privata della chiave di firma.

È necessario inserire una password lunga per evitare un attacco di forza bruta usando tutte le combinazioni di caratteri possibili.

!!! warning "Attenzione"
    Immetti una password, una frase, lunga almeno 12 caratteri.
