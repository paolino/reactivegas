# Help Reference

This page contains translated help content from the application's built-in help system.

## Purchasing Group

Select the purchasing group you wish to view.

## Administrator (Author of Declarations)

Select an administrator from those in the selected purchasing group.

The declarations published so far by the administrator are loaded into the session to allow their deletion and the addition of new ones.

## Declaration Management

This menu allows you to add new declarations to the session.

Additionally, in the last section you will find some management operations concerning the declarations themselves:

- **Publishing declarations** replaces the declarations previously published by the selected administrator with those currently in session
- **Deleting declarations** removes the declarations in session. The subsequent publication corrects those already published
- **Correcting the consideration level** of declarations allows you to observe an incomplete blank loading. This is necessary when we want to produce a declaration that has the right to be published, but loading all declarations prevents the system from offering it to us

## Effect of Latest Declarations

A *blank* calculation of the effects of declarations on the current state.

The inserted declarations include:

- Published declarations from all administrators except the selected one
- Session declarations from the selected administrator

## Administration

Administration menu for the service manager and for obtaining new keys.

## Password Entry

The password is used to hide the private part of the signing key.

It is necessary to enter a long password to prevent a brute force attack using all possible character combinations.

!!! warning
    Enter a password (a phrase) at least 12 characters long.
